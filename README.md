## 环境配置
### python3环境
```
pip install -r requirements.txt
```


## 使用方法
### 参数配置
main.py 内 run() 函数，自己根据需求配置参数  
可在文件"account.txt"中配置账号密码，也可在run()函数中直接配置cookie  
account.txt格式如下：  
```
23301999
123456
```
第一行为学号，第二行为密码  
### 主程序运行
```
python3 main.py
```
目前已经支持退课操作，全校任选课需要提前退课哦
